Sistem Penjadwalan Mata Kuliah

My group and I joined MBKM program at Faculty of Engineering, Jenderal Soedirman University to create course scheduling system integrated SIA using DevOps development method. The backend and api was built using django framework while for the frontend was built using react js. The system was completed in approximately five months.
