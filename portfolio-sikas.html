Sistem Informasi Pengelolaan Kas Kecil

Petty Cash Management Information System is a web-based system that has a function to manage petty cash in CV. Jenderal Software. This system is used for 1 user, namely admin. This system has functions for managing user data, managing account template data, managing years, managing master account data, managing journal data, and managing reports. Previously, petty cash management still used the manual method, which was written in a book. Based on existing problems, the authors try to build a Petty Cash Management Information System to facilitate computerized petty cash management. The system is created using the waterfall method and with a laravel framework. The system test results show that all functions on the system can run according to user needs.
